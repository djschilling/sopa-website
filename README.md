# SOPA Website

Minimal marketing/support website for **SOPA**, a puzzle game.

## Stack

- [Astro](https://astro.build/)
- TypeScript
- Plain CSS
- Static site output

## Pages

- `/` Home
- `/support` Support
- `/privacy` Privacy Policy
- `/terms` Terms of Use

## Development

```bash
npm install
npm run dev
```

Site runs locally at `http://localhost:4321` by default.

## Build

```bash
npm run build
npm run preview
```

Build output is generated in `dist/`.

## Deploy to Cloudflare

This repo is configured for a Cloudflare Worker static assets deploy using Wrangler.

1. In Cloudflare, create/connect the project to this GitHub repository.
2. Use `npm run build` as the build command.
3. Use `npm run deploy` as the deploy command.
4. Ensure your production branch is `main`.

Notes:
- `wrangler.toml` is configured to publish `dist/` as static assets.
- Build output is generated by Astro into `dist/`.
- If you attach a custom domain in Cloudflare, keep DNS and the `site` value in `astro.config.mjs` aligned.
